## 📞 Dúvidas Comuns

**P: Preciso de internet para abrir?**  
R: Sim, para carregar Tailwind CSS via CDN (< 100ms). Sem internet, estilos não carregam.

**P: Posso editar a página?**  
R: Sim! Abra em qualquer editor (VS Code, Notepad) e edite o HTML/CSS.

**P: Como fazer deploy?**  
R: Arraste o arquivo para Vercel ou Netlify (drag & drop) - deploy automático.

**P: Preciso de servidor próprio?**  
R: Não! É um arquivo HTML estático. Qualquer hospedagem funciona.

**P: Como adicionar mais agentes?**  
R: Copie um dos 6 cards de agente e customize (cor, ícone, texto).
